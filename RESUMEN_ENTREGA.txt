ARQUITECTURA TÉCNICA
================================================================================

STACK:
  Frontend:     Streamlit 1.28 (Python web)
  Backend:      Python 3.11
  Database:     SQLite3
  APIs:
    - yfinance (Precios)
    - Polymarket (Probabilidades)
    - NewsAPI (Noticias)
  Hosting:      Streamlit Cloud (Gratuito)
  Automation:   GitHub Actions
  Version:      Git/GitHub

FLUJO DE DATOS:

  [Usuario]
      ↓ (Navegador)
  [Streamlit Cloud - URL pública]
      ↓ (Lee)
  [SQLite - data/dashboard.db]
      ↑ (Escribe)
  [GitHub Actions - Scheduler]
      ↓ (Ejecuta)
  [Scripts Python]
      ↓ (Consultan)
  [yfinance, Polymarket, NewsAPI]

TABLAS DE BD:
  1. oil_prices (365 registros/año)
     - date, brent_price, wti_price, volatilidad, cambios

  2. polymarket_signals (4+ mercados, 90 días)
     - market_id, probability, liquidity, cambios

  3. news_events (30 días)
     - title, source, url, sentiment, impact, keywords

  4. risk_scores (agregado diario)
     - overall_score, componentes, nivel, cambios

  5. alerts_log (activas + histórico)
     - tipo, severidad, timestamp

  6. system_metadata (configuración interna)

PERFORMANCE:
  - Carga inicial: < 2 segundos
  - Actualización datos: < 10 segundos
  - Tamaño BD (365 días): < 10 MB
  - Usuarios simultáneos: 50+ (Streamlit Cloud)

FUNCIONALIDADES CLAVE
================================================================================

SCORE DE RIESGO SINTÉTICO (0-100)
  Componentes:
  ├─ Escalada (Polymarket)         [30%] ← Mayor peso
  ├─ Señal Precio Petróleo         [20%]
  ├─ Disrupciones Suministro       [20%]
  ├─ Volatilidad Petróleo          [15%]
  ├─ Sentimiento Noticias          [10%]
  └─ Severidad Geopolítica         [ 5%]

  Niveles:
  ├─ 0-30   → 🟢 BAJO (Monitoreo rutinario)
  ├─ 30-50  → 🟡 MEDIO (Vigilancia incrementada)
  ├─ 50-75  → 🟠 ALTO (Seguimiento diario)
  └─ 75-100 → 🔴 CRÍTICO (Evaluación urgente)

MÓDULOS DEL DASHBOARD

  Página 1: Resumen Ejecutivo
    ├─ Score principal (0-100) con cambios
    ├─ KPIs: Brent, WTI, volatilidad, escalada
    ├─ Descomposición del score
    ├─ Top 3 noticias del día
    ├─ Alertas activas
    └─ Mini gráfico Brent 7 días

  Página 2: Módulo Petróleo
    ├─ Gráfico histórico 12 meses (Brent + WTI)
    ├─ Estadísticas: cambios 1d, 7d, 30d, YTD
    ├─ Volatilidad 30 días rolling
    ├─ Spread Brent-WTI
    └─ Tabla histórica últimas 30 sesiones

  Página 3: Polymarket & Señales
    ├─ Tabla de mercados activos
    ├─ Categorías: escalada, suministro, logística, precio
    ├─ Probabilidades actuales + cambios
    ├─ Índice sintético de riesgo
    └─ Histórico individual por mercado

  Página 4: Noticias & Eventos
    ├─ Noticias de últimas 24-30 horas
    ├─ Filtros por impacto (bajo, medio, alto)
    ├─ Clasificación de sentimiento
    ├─ Links a fuentes originales
    └─ Estadísticas de cobertura

  Página 5: Análisis Avanzado
    ├─ Comparativa: hoy vs 1d, 7d, 30d, máx/mín
    ├─ Histórico de scores (gráfico)
    ├─ Desacoplamiento petróleo vs riesgo
    ├─ Alertas activas detalladas
    └─ Metodología documentada

AUTOMATIZACIÓN

  Daily (09:00 UTC):
    └─ Descarga precios petróleo (yfinance)

  Daily (10:00 UTC):
    └─ Descarga Polymarket + recalcula scores

  Cada 6 horas (00:00, 06:00, 12:00, 18:00 UTC):
    └─ Descarga Polymarket

  Cada hora (XX:00 UTC):
    └─ Descarga noticias + deduplica

  Auto (tras cada ingesta):
    └─ Recalcula scores y alertas

ALERTAS

  🔴 CRÍTICA:
    ├─ Brent sube/baja >5% en 1 día
    ├─ Score sube >20 puntos en 24h
    └─ Polymarket escalada >75%

  🟡 ADVERTENCIA:
    ├─ Brent sube/baja 3-5% en 1 día
    ├─ Score sube 10-20 puntos
    └─ Volatilidad >30%

  🔵 INFO:
    ├─ Nuevos eventos alto impacto
    └─ Desacoplamiento detectado

  6. Hacer push

  Hosting: Gratuito
  Uptime: 99.5%
  Usuarios simultáneos: 50+

OPCIÓN B: LOCAL (DESARROLLO)

  1. git clone <repo>
  2. python -m venv venv
  3. pip install -r requirements.txt
  4. echo "NEWSAPI_KEY=abc123" > .env
  5. python init_sample_data.py
  6. streamlit run app.py

  Resultado: Dashboard en http://localhost:8501
  Desarrollo: Inmediato
  Personalización: Total

OPCIÓN C: DOCKER (PRODUCCIÓN)

  1. docker build -t geopolitical-dashboard .
  2. docker run -e NEWSAPI_KEY="tu_clave" -p 8501:8501

  Resultado: Container listo para desplegar
  Escalabilidad: Múltiples instancias
  Hosting: Cualquier VPS/Kubernetes
